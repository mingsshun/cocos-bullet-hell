# Bullet Hell Game (Cocos Creator 3.7)

## 🎮 Demo
Link: https://mshdemo.netlify.app/3/

## 🎯 Features
- Player movement (WASD)
- Auto shoot
- 2 enemy types (chaser, shooter)
- Survival system (60s)
- Game over + restart
- Object pooling
- Event-driven architecture

## 🎮 Controls
- Move: WASD
- (optional) Dash: Space

## 🛠 Tech
- Cocos Creator 3.7
- TypeScript
- ECS + EventBus

## ⚙️ How to run
- Open bằng Cocos Creator
- Hoặc chạy build trong /build

## 📦 Build
- Desktop/Web build included

## 📌 Notes
- Code structured for extensibility
- No third-party packages used
